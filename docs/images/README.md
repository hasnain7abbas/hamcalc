# Screenshots

The main `README.md` references images in this directory. Drop your screenshots
here with the following filenames — they'll automatically appear in the README.

| File                          | What to capture                                                     |
|-------------------------------|---------------------------------------------------------------------|
| `hero.png`                    | The full app on desktop with a 2×2 Rabi matrix solved.              |
| `mobile.png`                  | Vertical screenshot on a phone showing matrix + keyboard.           |
| `keyboard.png`                | Close-up of the on-screen keyboard, ideally on the **Physics** tab. |
| `output-spectrum.png`         | Output panel on the **Spectrum** tab with eigenvalues.              |
| `output-eigenvectors.png`     | Output panel on the **Eigenvectors** tab.                           |
| `output-properties.png`       | Output panel on the **Properties** tab.                             |
| `output-charpoly.png`         | Output panel on the **Char Poly** tab.                              |
| `output-evolution.png`        | Output panel on the **U(t)** tab with a numeric matrix.             |
| `size-modal.png`              | The size-picker modal with the four presets.                        |

Recommended dimensions:
- Desktop screenshots: 1600 × 1000 (cropped to interesting area).
- Mobile: 390 × 844 (iPhone 14) or similar.
- PNG; transparent background not required.

Use the dark gradient backdrop the app already provides.
